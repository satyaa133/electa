# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/2305ef9b-4ec9-414d-ac24-bd0e93c4c5b2

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env` and set your API keys:
   ```sh
   cp .env.example .env
   # Edit .env and add your GEMINI_API_KEY and OAuth credentials
   ```
3. Run the app:
   `npm run dev`
